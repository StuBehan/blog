This is the first post. Everything below the `---` front matter block is just
**Markdown**, so you can write naturally.

## Writing a new post

Drop a file in `_posts/` named `YYYY-MM-DD-title.md`, give it front matter
like the block at the top of this file, write your content, then:

```bash
git add _posts/your-new-post.md
git commit -m "New post"
git push
```

The GitHub Actions workflow builds the site and publishes it automatically —
usually live within a minute.

## Previewing locally

```bash
bundle exec jekyll serve --livereload
```

Then open <http://localhost:4000>. Edits show up on save.
